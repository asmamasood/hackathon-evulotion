# Frontend Skills Index

## Overview
This directory contains specialized skills for implementing Next.js frontend systems. Each skill addresses a specific aspect of frontend development.

## Available Skills

### 1. Next.js App Router (`nextjs-app-router.md`)
Implement Next.js App Router structure with proper component organization and routing.

### 2. Responsive Design (`responsive-design.md`)
Implement responsive UI designs that work across all device sizes and screen resolutions.

### 3. Data Schema Design (`data-schema-design.md`)
Design table migration schemas and data structures for frontend-backend integration.

### 4. Component Architecture (`component-architecture.md`)
Structure reusable and maintainable React components with proper separation of concerns.

## Best Practices
- Always follow Next.js conventions and best practices
- Implement responsive design from the start
- Use TypeScript for type safety
- Follow accessibility standards
- Optimize for performance and user experience

## Integration Guidelines
- Each skill can be implemented independently
- Skills should work together to form a complete frontend system
- Follow consistent component architecture across all skills
- Maintain detailed documentation for each implementation
- Include comprehensive tests for all frontend functionality
