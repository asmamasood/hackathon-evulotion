# Database Skills Index

## Overview
This directory contains specialized skills for managing Neon Serverless PostgreSQL databases. Each skill addresses a specific aspect of database management.

## Available Skills

### 1. Schema Design (`schema-design.md`)
Design database schemas for scalability and maintainability in Neon Serverless PostgreSQL.

### 2. Migration Management (`migration-management.md`)
Perform database migrations safely and effectively in Neon Serverless PostgreSQL.

### 3. Query Optimization (`query-optimization.md`)
Optimize queries and indexes for performance in Neon Serverless PostgreSQL.

### 4. Neon Serverless Integration (`neon-integration.md`)
Integrate effectively with Neon Serverless PostgreSQL features and capabilities.

## Best Practices
- Always follow PostgreSQL and Neon Serverless best practices
- Maintain data integrity and consistency
- Implement proper security measures
- Use appropriate indexing strategies
- Monitor database performance regularly

## Integration Guidelines
- Each skill can be implemented independently
- Skills should work together to form a complete database management system
- Follow consistent schema design patterns across all skills
- Maintain detailed documentation for each implementation
- Include comprehensive testing for all database operations
